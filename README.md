### 📦 我的开源项目

- **[MCGA](https://github.com/JiaGuZhuangZhi/MCGA)**  
  > 为 ColorOS 用户打造的轻量 LSPosed 模块，支持 Dock 模糊、隐藏应用名、AOD 全景沉浸等个性化定制。  
  *技术栈：Kotlin · Xposed · Jetpack Compose · haze*

- **[Gustate-AppBar](https://github.com/JiaGuZhuangZhi/Gustate-AppBar)**  
  > 一个支持背景模糊、系统栏适配、动态透明度的 Android 自定义 AppBar，开箱即用。  
  *技术栈：Kotlin · BlurView · ViewCompat · XML 属性扩展*

- **参与 [UOTAN for Android](https://github.com/Uotan-Dev/UOTAN-for-Android)**  
  > 柚坛社区官方安卓客户端，共建开源论坛生态。
